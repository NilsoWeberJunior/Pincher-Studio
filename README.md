# PincherAnim
O PincherAnim é um site onde você pode criar suas animações 100% de graça e sem anúncios! salvando sua animação em mp4!

## Como usar?

para usar o PincherAnim basta ir no html inicial (index.html) e clicar na logo do pincher anim, depois ir em +🎦 para começar a criar sua animação...
