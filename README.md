# Pincher Studio
O Pincher Studio é um site onde você pode criar suas animações e pinturas 100% de graça e sem anúncios! salvando sua animação em mp4 e sua pintura em png (com trasparencia ou não! você escolhe!)

## Como usar?

para usar o Pincher Studio basta ir no html inicial (index.html) e você pode clicar em uma das opções que aparecem na barra! (criar animação/criar pintura)
